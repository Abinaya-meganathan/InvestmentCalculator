📌 Investment Calculator (SIP & Lump Sum)

A simple web-based Investment Calculator built using ASP.NET Core MVC.
This application helps users calculate returns for SIP (Systematic Investment Plan) and Lump Sum investments.

The project is designed to be beginner-friendly, easy to understand, and visually pleasant with a clean green-themed UI inspired by popular finance apps.

✨ Features

Calculate SIP (Monthly Investment) returns

Calculate Lump Sum investment returns

Inputs:

Investment amount

Interest rate (%)

Investment duration (years)

Displays:

Total Invested Amount

Final Investment Value

Total Profit

Clean, responsive, and user-friendly UI

No login, no database – simple and fast

🛠 Technologies Used

ASP.NET Core MVC

C#

Razor Views

Bootstrap 5

Custom CSS

HTML

🎨 UI & Design

Responsive layout using Bootstrap

Soft green theme inspired by modern fintech applications

Clean card-based design

Focus on simplicity and readability

🧠 How It Works (In Simple Terms)

The user enters investment details on a single page.

The controller calculates the investment returns based on user input.

The result is displayed on the same page without reloading the app structure.

The project follows the MVC (Model–View–Controller) pattern.
